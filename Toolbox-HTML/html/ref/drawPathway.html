Toolbox Package Symbol

drawPathway

drawPathway[metPos, rxnPos, labels]
will draw a pathway map from the provided species, reactions, and text labels.
MORE INFORMATION
  • The expression metPos must be of the following form {id -> {x, y}, ...}.
  • The expression rxnPos must be of the following form {id -> {{{pt1, pt2, ...}, direction},...}, ...} specifying bezier curve or line elements, where direction is either 0 (no arrow heads), 1 (arrowheads in the forward direction), or -1 (for arrowheads in the reverse direction).
  • labels
  • The following options can be given:
ReactionData{}reaction centric data
MetaboliteData{}metbolite centric data
  • The options ReactionData and MetaboliteData can be used visualize
