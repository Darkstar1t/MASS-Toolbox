Toolbox Package Symbol

str2mass

str2mass["expression"] parses string representations of metabolites, reactions, rate and equilibrium constants and other parameters.
MORE INFORMATION
  • The following options can be given:
ReversibleDelimiterRegularExpression[<[=-]*>]Provide a regular expression that matches a reversible reaction delimiter
IrreversibleDelimeterRegularExpression[[=-]*>]Provide a regular expression that maches a irreversible reaction delimeter
