Constant

Constant
is a list attribute containing species in the model who should be treated as constant.

Access the constant species of a model
The glucose exchange reactions and its associated rate equation (glucose concentration is time dependent)
Set new constant species for model (discard old ones)
Glucose concentration is not time dependent anymore
Updating the constant attribute will add to the already existing constant species.
