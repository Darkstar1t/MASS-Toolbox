enzyme

enzyme["ID"->"id"]
represents an enzyme.
MORE INFORMATION
  • The following options can be given:
IDBurbThe id of the enzyme
Compartment_The compartment of the enzyme
BoundCatalytic{}Species that are bound to the enzyme and take part of the catalytic process
BoundActivators{}A list of allosterically bound activators
BoundInhibitors{}A list of allosterically bound inhibitors
CatalyticSitesInfinityThe number of catalytic sites
ActivationSitesInfinityThe number of allosteric activation sites
InhibitionSitesInfinityThe number of allosteric inhibition sites
