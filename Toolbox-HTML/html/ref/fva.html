fva

fva[model]
performs flux-variability analysis (FVA) on the given system.
MORE INFORMATION
  • The following options can be given:
OptFlagMax"Min" (Minimization) or "Max" (Maximization)
SolverLinearProgrammingChoose from all supported solvers, i.e. LinearProgramming, GurobiSolve, GLPKStandalone, CPLEXStandalone
LooplessFalseSet True if loopless optimization should be performed
ProgressBarTrueIf a progress should be shown
