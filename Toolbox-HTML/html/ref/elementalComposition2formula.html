Toolbox Package Symbol

elementalComposition2formula

elementalComposition2formula[elementalComposition]
will translate the elemental composition (e.g., {"O"->1, "H"->2}) into a molecular formula (H2O)
MORE INFORMATION
  • The following options can be given:
PseudoElementsTrueIf unknown characters should be treated as pseudo elements
EscapingCharacter&The character to escape pseudo elements
EXAMPLESCLOSE ALL
