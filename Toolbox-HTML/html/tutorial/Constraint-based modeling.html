Flux-balance and flux-variability analysis

Div[{class -> Text}, {Flux-balance analysis (FBA) finds a steady-state flux solution that maxmizes a cellular objective, e.g., optimal growth rate. For example, FBA can be used to asses the consequence of genetic perturbations. Flux-variablity analysis (FVA) calculates effective flux bounds by minimizing and maximizing flux through individual reactions. For example, FBA solutions are rarely unique and FVA can be used to assess the space of alternativa optimal solutions.}]Div[{class -> MathTable, id -> DefinitionBox}, {XMLElement[table, {class -> Table, width -> 100%, cellpadding -> 5, style -> border-collapse: collapse;}, {XMLElement[col, {width -> 40%}, {}], XMLElement[col, {width -> 60%}, {}], XMLElement[tr, {class -> lineAbove, style -> }, {XMLElement[td, {class -> IconColumn}, {fba, [, model, ,, , obj, ]}], XMLElement[td, {}, {XMLElement[span, {class -> TT, id -> TableText}, {find a steady-state flux distribution that maximizes obj}]}]}], XMLElement[tr, {class -> , style -> }, {XMLElement[td, {class -> IconColumn}, {fva, [, model, ]}], XMLElement[td, {}, {XMLElement[span, {class -> TT, id -> TableText}, {find minimum and maximum fluxes for all reactions in model}]}]}]}]}]
