Getting started

The toolbox is not yet publicly available (and this site is under construction). It has been submitted for publication. If you're one of the peer reviewers you can access the toolbox at the currently private Github repository with the following credentials

  • username: toolbox-reviewer
  • password: np-P13xxxxA (the four x's are the last four digits of the manuscript number)

    • The repository contains all the developer source code (that will be made open source after publication). The README.md file (top level directory) contains a URL to a version of the toolbox intended for installation. It contains the compiled documentation and an installation notebook (Installer.nb).

    If you don't have access to Mathematica we can recommend the 30 day unrestricted trial version provided by Wolfram.
